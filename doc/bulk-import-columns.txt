(ถ้ามีไฟล์เก่า bulk-import-schools-sample.csv ที่ข้อมูลยาวหรือสับสน — ปิดไฟล์ในโปรแกรมแล้วลบหรือแทนที่ได้)
รูปแบบ: CSV UTF-8 (เปิดใน Excel ได้ — ถ้าภาษาไทยเพี้ยนให้ Import จาก CSV แล้วเลือก UTF-8)

แถวแรก = หัวคอลัมน์ (ห้ามลบ / ห้ามสลับลำดับถ้าใช้สคริปต์นำเข้าตามตัวอย่างนี้)

แถวถัดไป = หนึ่งแถวต่อหนึ่งโรงเรียน

--- โรงเรียน (คอลัมน์ 1–14) ---
id              ว่างได้ = ถ้า name ตรงกับโรงเรียนเดิมจะอัปเดตโรงเรียนนั้น/คะแนนแทน | ถ้าไม่เจอชื่อจะสร้างใหม่ | ใส่ UUID = อัปเดตโรงเรียนเดิม
name            ชื่อโรงเรียน (ไทย) บังคับ (ระบบสร้างคีย์ normalize จากชื่อเพื่อจับคู่ซ้ำและ unique — ควรใช้การสะกดสม่ำเสมอ)
nameEn          ชื่ออังกฤษ (ว่างได้)
province        จังหวัด บังคับ
affiliation     สังกัด บังคับ (เช่น สพฐ. อปท. เอกชน สช.)
level           ระดับการศึกษา บังคับ (เช่น ประถมศึกษา มัธยมศึกษา)
website         ว่างได้
contact         ว่างได้
description     ว่างได้
address         ว่างได้ | ถ้ามีเครื่องหมายจุลภาค ให้ครอบทั้งช่องด้วย "..."
phone           ว่างได้
facebookUrl     ว่างได้
lineId          ว่างได้
isPublished     0 = ไม่เผยแพร่ | 1 = เผยแพร่ในอันดับสาธารณะ

--- คะแนน (คอลัมน์ 15–39) = a1 ถึง e5 ตามแบบประเมิน ---
ค่าแต่ละช่อง 0–4 (ทศนิยมได้ถ้าระบบรองรับ) รวม 25 ช่องต่อแถว

แถวตัวอย่างแถวที่ 1 ใช้รูปแบบคะแนนง่ายๆ:
  หมวด A = 5 ช่อง ได้ 3 ทั้งหมด | B = 3 ทั้งหมด | C = 2 ทั้งหมด | D = 1 ทั้งหมด | E = 0 ทั้งหมด
แถวตัวอย่างแถวที่ 2 ใช้คะแนน 2 ทุกช่อง (ทั้ง 25 ช่อง)
